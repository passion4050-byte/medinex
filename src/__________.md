### 방법 2: 수동으로 파일 복사 (백업용)

버튼을 못 찾았다면, 파일을 하나씩 다운로드:

#### Windows 사용자:

```
1. 바탕화면에 새 폴더 만들기
   우클릭 > 새로 만들기 > 폴더
   이름: medinex-landing

2. 폴더 안에 하위 폴더 만들기:
   - components
   - styles
   - src

3. 각 파일 내용 복사:
   이 화면에서 파일 클릭 > 내용 복사
   메모장에 붙여넣기 > 같은 이름으로 저장
```

#### Mac 사용자:

```
1. Finder 열기
2. 문서 폴더에 새 폴더 만들기
   이름: medinex-landing

3. 폴더 안에 하위 폴더 만들기:
   - components
   - styles
   - src

4. 각 파일 내용 복사:
   이 화면에서 파일 클릭 > 내용 복사
   텍스트 편집기에 붙여넣기 > 같은 이름으로 저장
```

## 🌐 Step 2: GitHub에 올리기

### 웹브라우저로 하기 (코딩 몰라도 OK!)

**1. GitHub 회원가입**

```
https://github.com/signup
이메일, 비밀번호 입력
무료 계정으로 가입
```

**2. 새 저장소 만들기**

```
https://github.com/new 접속

┌─────────────────────────────┐
│ Repository name:             │
│ medinex-landing              │  ← 입력
│                              │
│ Description:                 │
│ MEDINEX 랜딩 페이지          │  ← 입력 (선택)
│                              │
│ ⚫ Public  ⚪ Private         │  ← Public 선택
│                              │
│ [Create repository] 버튼 클릭 │
└─────────────────────────────┘
```

**3. 파일 업로드**

```
새로 만든 저장소 페이지에서:

"uploading an existing file" 링크 클릭

┌─────────────────────────────┐
│ Drag files here to add them │
│                              │
│   Step 1에서 준비한 모든     │
│   파일을 여기로 드래그!      │
│                              │

파일 선택 방법:
1. 폴더 열기
2. Ctrl+A (모두 선택) - Windows
   Cmd+A (모두 선택) - Mac
3. 드래그해서 GitHub로 끌어놓기

아래에 메시지 입력:
```

```
┌─────────────────────────────┐
```

```
│ Updated 2 minutes ago        │
```

## 🎯 Step 4: 아임웹에 붙이기

### 마지막 단계!

**1. 아임웹 관리자 접속**

```
https://admin.imweb.me 접속
로그인
```

**2. 새 페이지 만들기**

```
좌측 메뉴:
디자인 > 페이지

┌─────────────────────────────┐
│ [+ 새 페이지 만들기]          │  ← 클릭
└─────────────────────────────┘

페이지 정보:
┌─────────────────────────────┐
│ 페이지 이름: MEDINEX         │  ← 입력
│                              │
│ 템플릿: 빈 페이지            │  ← 선택
│                              │
│ [확인]                       │  ← 클릭
└─────────────────────────────┘
```

**3. HTML 위젯 추가**

```
페이지 편집 화면에서:

좌측 위젯 목록:
┌─────────────────────────────┐
│ 🔍 검색: HTML                │
└─────────────────────────────┘

HTML 위젯을 찾아서
→ 페이지 중앙으로 드래그

HTML 위젯을 더블클릭
```

**4. 코드 붙여넣기**

```
HTML 편집기가 열리면:

아래 코드를 복사해서 붙여넣기:
```

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <style>
    body, html {
      margin: 0;
      padding: 0;
      overflow-x: hidden;
    }
    .medinex-iframe {
      width: 100%;
      height: 4500px;
      border: none;
      display: block;
    }
    @media (max-width: 768px) {
      .medinex-iframe {
        height: 6000px;
      }
    }
  </style>
</head>
<body>
  <iframe
    class="medinex-iframe"
    src="여기에_Vercel_URL_붙여넣기"
    scrolling="no"
  ></iframe>
</body>
</html>
```

**⚠️ 필수: URL 변경하기**

```
찾기: src="여기에_Vercel_URL_붙여넣기"

변경:
src="https://medinex-landing-abc123.vercel.app"
     ↑ Step 3에서 복사한 실제 URL
```

**5. 저장 & 발행**

```
HTML 편집기:
[저장] 클릭

우측 상단:
⚙️ (설정) 클릭
→ ☑️ 헤더 숨김
→ ☑️ 푸터 숨김
→ [저장]

[발행] 버튼 클릭
```

**✅ 완료!** 🎉

## ❓ 문제가 생겼나요?

### 흔한 문제 해결:

**문제 1: Vercel 빌드 실패**

```
원인: package.json 파일 누락

해결:
1. GitHub 저장소 확인
2. package.json 파일이 있는지 확인
3. 없으면 다시 업로드
```

**문제 2: 아임웹 페이지가 비어있음**

```
원인: URL 오타

해결:
1. HTML 코드 다시 확인
2. src="..." 부분 확인
3. Vercel URL이 정확한지 확인
4. https:// 포함되었는지 확인
```

**문제 3: 높이가 안 맞음**

```
해결:
HTML 코드에서 숫자 변경:
height: 4500px → height: 5000px
```
